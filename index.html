What is Grid CSS

This CSS file is built upon Bootstrap 5's Grid System.

Grid System CSS is a powerful layout system that allows you to create responsive designs with ease. It provides a flexible grid of columns and rows that can be easily customized to fit your needs. With the Grid System CSS, you can create complex layouts by combining different column classes and adding offsets to create spacing between columns. It is a great tool for building modern and responsive websites.

How to use

To use Grid CSS, you need to include the CSS file in your HTML file. Then, you can use the grid system by adding the class name to the HTML element.

For example, if you want to create a 3-column layout, you can use the class name .col-4 for each column.

You can also use the class name .offset-* to add space between columns.
